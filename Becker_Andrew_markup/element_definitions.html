HTML Element Definitions

<h1>
Main heading.
<h2>
Level 2 heading
<h3>
Level 3 heading
<h4>
Level 4 heading
<h5>
Level 5 heading
<h6>
Level 6 heading
<p>
A paragraph
<b>
Makes words appear bold
<i>
Makes words appear italic
<sup>
Superscript
<sub>
Subscript
<br />
Creates a line break
<hr />
Creates a horizontal rule
<strong>
Shows strong importance
<em>
Shows emphasis
<blockquote>
Used for larger quotes that take up an entire paragraph
<q>
Adds inline quotations
<abbr>
Abbreviates words or creates acronyms
<cite>
Indicates where the citation is from
<dfn>
Used to define the instance of a new term
<address>
Used to contain contact details for the author of the page
<ins>
Used to show content has been inserted
<del>
Shows text has been deleted
<s>
Strikethrough to show something is no longer accurate or relevant
<ol>
Creates an ordered list
<li>
Each item in a list is placed between an opening and closing li tag
<ul>
Creates an unordered list
<dl>
Creates a definition list. Usually a series of terms and definitions
<dt>
Contains the term for the definition list
<dd>
Used to contain the definition in a definition list
